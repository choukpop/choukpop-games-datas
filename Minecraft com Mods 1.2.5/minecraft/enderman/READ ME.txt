Summarizing: This mod goes inside minecraft.jar :-)

ModLoader/forge are not required but are compatible.

How to install:
1) Open the directory %APPDATA%
2) Open the directory .minecraft
3) Open the directory bin
4) Open the file minecraft.jar with WinRAR
5) Drag and drop the files inside this zip file to minecraft.jar
